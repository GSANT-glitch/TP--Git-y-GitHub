# Desarrollo de Sistemas de Inteligencia Artificial 2026 1C
## Ejercicios de Data Science con Python

Este repositorio contiene la resolución de tres ejercicios prácticos para la introducción a la manipulación y análisis de datos utilizando Python.

## 📝 Contenido de los Ejercicios

### Ejercicio 1: Importación de Librerías y NumPy
* Importación de librerías esenciales (`numpy`, `pandas`, `seaborn`, `matplotlib`).
* Creación y visualización de un array unidimensional de 5 elementos con NumPy.

### Ejercicio 2: Manipulación de Datasets con Pandas
* Creación de un DataFrame con calificaciones de estudiantes.
* Operaciones estadísticas básicas:
  * Visualización de la tabla de datos.
  * Cálculo de la nota promedio.
  * Obtención de la nota máxima y mínima.

### Ejercicio 3: Criterio de Selección de Herramientas
Justificación breve de las librerías ideales para tareas específicas de Data Science:
1. **Pandas:** Lectura de archivos CSV y manejo de datos tabulares.
2. **Seaborn / Matplotlib:** Creación de gráficos y visualización de distribuciones.
3. **Scikit-learn:** Desarrollo y entrenamiento de modelos de Machine Learning.
4. **NumPy:** Operaciones matemáticas avanzadas y cálculo matricial eficiente.

## 🛠️ Tecnologías Utilizadas
* Python 3
* NumPy
* Pandas
* Matplotlib
* Seaborn
